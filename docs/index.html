Frequently Asked Questions

How do I find my Epson printer's IP address?

From the printer: Go to Settings > Network Settings > Print Network Status. The IP will be printed on the sheet.

From your router: Log into your router's admin page and look for connected devices. The printer usually shows up as "EPSON" followed by a model number.

From the Epson app: Open the Epson iPrint app on your phone — the printer IP is shown in the printer settings.

What does "waste ink pad full" actually mean?

During print head cleaning cycles, excess ink flows into absorbent pads inside the printer. Epson tracks this with a software counter. When the counter exceeds a threshold (maintenance level > 94), the printer locks up and refuses to print.

The counter is deliberately conservative — most printers are locked long before the pads are actually saturated. This tool resets the counter back to zero.

Is this safe? Will it damage my printer?

The tool only changes the waste ink counter values in the printer's EEPROM — the same values that Epson's own service tool resets. It does not modify firmware or any other settings.

One caveat: if your waste ink pads are genuinely saturated (you see ink leaking from the bottom of the printer), you should clean or replace the pads before resetting the counter.

Do I need to install Python?

No. The Windows .exe and macOS .app downloads include everything needed. Just download and run.

If you prefer to run from source (or you're on Linux), you'll need Python 3.8+. No pip packages required — the tool uses only the Python standard library.

Does my computer need to be connected via USB?

No. This tool works entirely over WiFi. Your computer and printer just need to be on the same local network. The tool communicates via SNMP (UDP port 161).

Which Epson printers are supported?

Currently tested and confirmed on the Epson ET-2720. It should also work on ET-2721 and ET-2700 as they share the same EEPROM layout and SNMP keys.

Other EcoTank models may use different EEPROM addresses. If you have a different model and want to help, please open an issue on GitHub.

My firewall is blocking it. What port does it use?

The tool sends and receives UDP packets on port 161 (SNMP). Make sure your firewall allows outbound UDP traffic to your printer's IP on port 161.

Is this legal?

Yes. You own your printer and have the right to repair it. This tool uses standard network protocols (SNMP) to modify settings on hardware you own. No encryption or DRM is bypassed. Right-to-repair laws in the US, EU, and Australia explicitly protect this kind of activity.
